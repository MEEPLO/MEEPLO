## 시작

나 같은 경우에는 react native cli로 설치를 진행했을 때 새 버전 0.69.0에서 흔히 나타나는 오류가 떴다.

```
TypeError: cli.init is not a function
```



그래서 `npx react-native init ProjectName --version 0.68.2` 로 설치
